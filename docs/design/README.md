# Design Docs Index

Use this naming convention for new design documents:

`docs/design/NNN-short-kebab-title.md`

Rules:
- `NNN` is a zero-padded sequence (`001`, `002`, ...)
- `short-kebab-title` is concise and specific
- one major design topic per file

Examples:
- `docs/design/000-curriculum-roadmap.md`
- `docs/design/001-minimal-agent-loop.md`
- `docs/design/002-llm-adapter-layer.md`
- `docs/design/003-memory-model.md`
- `docs/design/004-action-model.md`
- `docs/design/005-tool-interface-and-registry.md`
- `docs/design/006-planner-architecture.md`
